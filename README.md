# CortiScope AI 🌸

CortiScope AI is an intelligent therapeutic assistant designed to support emotional well-being using CBT techniques, stress detection, and personalized AI responses.

## ✨ Features

* 💬 AI-powered mental health conversations
* 🧠 Context-aware emotional analysis
* 🌿 Stress & cortisol-based insights
* 🚨 Crisis detection with support resources

## ⚙️ Tech Stack

* Frontend: React.js
* Backend: Node.js + Express
* AI: Groq API (LLaMA 3.1)

## 🚀 How to Run Locally

### 1. Clone the repository

git clone https://github.com/Vishnupriyaa0109/cortiscope.git
cd cortiscope

### 2. Backend setup

cd backend
npm install
node server.js

### 3. Frontend setup

cd ..
npm install
npm start

## 🔐 Environment Variables

Create a `.env` file inside `backend/`:

GROQ_API_KEY=your_api_key_here
PORT=5000

## 📌 Note

This project is for educational purposes and provides emotional support, not medical advice.
